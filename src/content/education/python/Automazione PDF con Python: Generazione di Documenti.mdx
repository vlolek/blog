## 📄 1. Generazione Automatica di PDF

In queste sezioni ho imparato a manipolare i file PDF. Non si tratta solo di scrivere testo, ma di gestire il layout, le celle e l'automazione partendo da file esterni (come i CSV).

**Cosa ho imparato:**

- Utilizzo della libreria `fpdf`.
- Creazione di header e footer dinamici.
- Iterazione su file Excel/CSV per generare fatture massive.

### Esempio Pratico

Ecco come genero un PDF di base con una riga di intestazione:

```python
from fpdf import FPDF

pdf = FPDF(orientation="P", unit="mm", format="A4")
pdf.add_page()
pdf.set_font(family="Arial", style="B", size=16)
pdf.cell(w=0, h=12, txt="Report Mensile Vendite", align="L", ln=1)
pdf.line(10, 21, 200, 21)

pdf.output("report.pdf")
```
